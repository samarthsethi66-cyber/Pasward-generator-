# 🔐 Advanced Password Generator

A modern and secure **Password Generator Web App** built using **HTML, CSS, and JavaScript**.
This project allows users to generate strong and customizable passwords instantly with a clean and responsive user interface.

## ✨ Features

* 🔐 Generate **strong and secure passwords**
* 🎛 **Adjustable password length** (6 – 32 characters)
* 🔠 Include **uppercase letters**
* 🔡 Include **lowercase letters**
* 🔢 Include **numbers**
* 🔣 Include **symbols**
* 📊 **Password strength indicator**
* 📋 **One-click copy to clipboard**
* 🎨 Modern and responsive UI design

## ⚙️ How It Works

1. Select the **password length** using the slider.
2. Choose the **character types** you want:

   * Uppercase
   * Lowercase
   * Numbers
   * Symbols
3. Click **Generate Password**.
4. Copy the password instantly using the **Copy button**.
